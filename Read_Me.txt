Overview
This project is a responsive, interactive dashboard built using HTML, CSS, JavaScript, and Chart.js. It visualizes quarterly revenue data across four regions (North, South, East, West) and allows users to dynamically filter by region and switch between chart types.

Features
Region Filter: View revenue data for all regions or a selected region.

Chart Type Switcher: Toggle between Bar, Line, Pie, and Doughnut charts.

Responsive Design: Optimized for various screen sizes.

Modular Code Structure: Clean separation of HTML, CSS, and JavaScript.

File Structure
File	Description
index.html	Main HTML file that loads the dashboard and links to external resources.
style.css	Contains all styling rules for layout, typography, and responsiveness.
script.js	Handles chart rendering, user interactions, and dynamic updates.

Dependencies
Chart.js CDN: https://cdn.jsdelivr.net/npm/chart.js : Used for rendering charts.
No additional installation is required. All dependencies are loaded via CDN.

How to Run
Ensure all three files (index.html, style.css, script.js) are in the same directory.
Open index.html in any modern web browser.
Use the dropdowns to explore different regions and chart types.
